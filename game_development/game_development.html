Game Development

Client Side

In game development, the client side refers to the player's software or hardware device, or both, that they use to play the game. The client side is responsible for translating the player's inputs into messages that are sent to a server for processing.

Game Mathematics

Mathematics is used in game development to model the behavior of objects in the game world, and to create lifelike interactions between characters, vehicles, and the environment. Some mathematical concepts used in game development include: Kinematics, Newtonian mechanics, and Calculus.

Game Physics

Computer animation physics or game physics are laws of physics as they are defined within a simulation or video game, and the programming logic used to implement these laws. Game physics vary greatly in their degree of similarity to real-world physics.

Game Engine

A game engine is a software development environment that helps users create video games more easily. Game engines provide a framework that includes tools for animation, physics, artificial intelligence, and audio.
